# Terminal Media Library

A simple terminal-based media library to manage **shows, games, and manga**.  
Data is stored in a **JSON database**, and you can **add, edit, or remove items** directly from the terminal.

---

## Features

- Manage **shows, games, and manga** in a single library.
- Add new items with metadata (title, genre, status, etc.).
- Edit existing items to update details.
- Remove items you no longer want in the library.
- Persistent storage with a JSON database.
